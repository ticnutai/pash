# Icon variants (לבחירה)

פתח/י כל קובץ SVG בתיקייה הזו כדי לראות תצוגה.

- `icon-variant-01-bookmark.svg` — ספר פתוח + סימנייה זהב (קלאסי/קריא).
- `icon-variant-02-scroll.svg` — מגילה עם ידיות זהב (אסוציאציה חזקה לתורה).
- `icon-variant-03-tablets.svg` — לוחות + טבעת זהב (אייקון "חזק" וברור בחנות).
- `icon-variant-04-minimal-book.svg` — קווי מתאר מינימליים (נקי, מודרני).
- `icon-variant-05-book-shield.svg` — מגן זהב עם ספר בפנים ("רשמי"/פרימיום).
- `icon-variant-06-parchment-mark.svg` — רקע קלף + מסגרת נייבי + כוכב/סימון זהב (בהיר ושונה).
- `icon-variant-07-white-frame-torah.svg` — עיגול לבן (פינות שקופות) + מסגרת זהב דקיקה + טקסט "תורה" עדין בנייבי.

וריאציות טקסט קצר יותר (אותו עיצוב בדיוק):
- `icon-variant-07a-circle-letter-t.svg` — טקסט: "ת".
- `icon-variant-07b-circle-to.svg` — טקסט: "תו".
- `icon-variant-07c-circle-chumash.svg` — טקסט: "חומש".
- `icon-variant-07d-circle-torah-compact.svg` — טקסט: "תורה" אבל דחוס/קומפקטי יותר.

אם תבחר/י אחת, אוכל להמיר אותה לסט אייקונים מלא (Play Store 512×512 + Android mipmap/adaptive + PWA icons) באותו סגנון.
