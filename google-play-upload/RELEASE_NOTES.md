# הערות גרסה (Release Notes)

## גרסה 1.4.1 — עברית

```
תיקון תצוגה חשוב באנדרואיד

מה הייתה הבעיה:
• במכשירים מסוימים (Edge-to-Edge), אייקוני המערכת העליונים והכפתורים הצפים התחתונים חפפו חלקים בממשק.

מה תוקן:
• תוקן Safe Area בחלק העליון כדי למנוע חפיפה עם שורת הסטטוס.
• תוקן Safe Area בחלק התחתון כדי למנוע חפיפה עם סרגל הניווט.
• שופרה יציבות מיקום הכפתורים הצפים במסכים ניידים.
```

## Version 1.4.1 — English

```
Important Android display fix

What was the issue:
• On some edge-to-edge Android devices, top system icons and bottom floating buttons could overlap parts of the app UI.

What was fixed:
• Top safe-area handling was fixed to prevent status bar overlap.
• Bottom safe-area handling was fixed to prevent navigation bar overlap.
• Floating button positioning was stabilized on mobile screens.
```
